**TeamLead**：主对话 Agent

## 需求概述

在用户登录后的界面中显示账户剩余额度信息，数据从 Codex 官方 API 获取。

## 核心功能

### 额度类型
- **5小时限额**：每5小时的使用限制和剩余量
- **周限额**：每周的使用限制和剩余量

### 显示内容
- 5小时限额的剩余量和百分比
- 周限额的剩余量和百分比
- 下次重置时间（5小时后、本周结束）
- 可选：已使用量

### 显示位置（待探索确认）
- 顶部状态栏
- 设置页面的账户信息区域
- 用户头像下拉菜单
- 其他合适位置

### 更新频率（待探索确认）
- 登录时获取一次
- 定时自动刷新（建议每分钟或每次对话后）
- 用户手动刷新按钮
- 接近限额时更频繁刷新

### 交互行为（待探索确认）
- 额度不足时是否需要警告提示
- 是否需要点击查看详细信息
- 是否需要跳转到官方充值页面的入口
